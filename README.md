# POC: Redis

It demonstrates how to run commands to manipulate data from [Redis](https://github.com/antirez/redis) running on [Docker](https://www.docker.com/) using [TestContainers](https://github.com/testcontainers).

## How to run

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
